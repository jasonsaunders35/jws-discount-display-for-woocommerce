Displays each sale product's dollar or percent discount.

== Description ==
This plug-in for WooCommerce is designed to emphasize the value of sale items by displaying their relative (percent) or absolute (dollar) discounts in a visual, customizable way. The backend includes a real-time preview of how several powerful, but easy-to-use style settings will appear.  It will be displayed anywhere product thumbnails are shown including categories, search results, related products, up-sells, and cross-sells.  Optionally, it can appear in product detail sections.

== Installation ==

1. Settings located at WooCommerce > Discount Display
2. Disabled by default. Must be enabled by using the 'Enabled' setting
